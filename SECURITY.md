# Security

## Reporting

If you find a security issue in the **template** (skills, scripts, or documentation), open a private report via [GitHub Security Advisories](https://github.com/IvanGomezMartinez/MatStudyLab/security/advisories/new) or contact the maintainer through their GitHub profile.

## Scope

MatStudyLab is a local workflow template. It does not run a hosted service.

## User responsibility

- Do **not** commit patient data, client identifiers, or proprietary measurement files to a public fork.
- Keep laboratory scripts and `LORE.md` in private clones when they contain sensitive context.
- Review AI-generated MATLAB before running it on production measurement data.
